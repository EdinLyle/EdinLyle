- **安全工具开发**：Python / Tkinter GUI / Java 工具，自动化脚本与漏洞扫描器
- **恶意软件分析**：静态 + 动态分析、病毒样本取证、蠕虫 / 挖矿专项处置
- **代码审计 (Code Review)**：Fortify 自动化审计 + 手工审计，Java / Python 安全开发
